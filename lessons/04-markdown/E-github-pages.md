## What Is GitHub Pages?

GitHub Pages lets you host a static website directly from a repo.
It is perfect for:

- Portfolios
- Project documentation
- Simple blogs

<br><br><br><br><br>

## Simple Setup (One Repo)

1) Create a repo on GitHub
2) Add an `index.md` or `index.html`
3) Push the repo to GitHub

<br><br><br><br><br>

## Enable Pages

On GitHub:

- Open the repo
- Go to **Settings** -> **Pages**
- Select the branch (usually `main`)
- Choose the root folder
- Save

After a minute, GitHub will give you a public URL.

<br><br><br><br><br>

## Pro Tip: Use Markdown

If you create `index.md`, GitHub Pages can render it into HTML.
That means you can publish a site using only Markdown.

<br><br><br><br><br>

## Summary

GitHub Pages turns a repo into a website.
It is a simple way to publish your work.
