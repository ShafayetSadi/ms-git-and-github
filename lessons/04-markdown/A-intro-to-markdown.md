## What Is Markdown?

Markdown is a **lightweight formatting language**.
It lets you write plain text that can be rendered as rich formatted content.

You type simple symbols like `#` and `-`, and tools turn them into headings and lists.

<br><br><br><br><br>

## Why Developers Love Markdown

- Easy to read in raw text
- Easy to write without a mouse
- Works everywhere (GitHub, editors, docs)
- Converts cleanly to HTML and PDF

It is the default language for README files and documentation.

<br><br><br><br><br>

## Example (Raw Markdown)

```
# My Project

This project is **fast**, *simple*, and easy to use.

## Features
- Works offline
- No setup needed
```

When rendered, it becomes a nicely formatted document.

<br><br><br><br><br>

## Where You Will See Markdown

- README.md files
- GitHub issues and pull requests
- Docs sites
- Note apps like Obsidian

If you use GitHub, you already use Markdown.

<br><br><br><br><br>

## Summary

Markdown is plain text with simple formatting rules.
It is the standard way developers write documentation.
