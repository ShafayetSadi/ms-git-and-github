## Headings

```
# H1
## H2
### H3
```

<br><br><br><br><br>

## Emphasis

```
**bold**
*italic*
```

<br><br><br><br><br>

## Lists

Unordered:

```
- item one
- item two
```

Ordered:

```
1. first
2. second
```

<br><br><br><br><br>

## Links and Images

```
[OpenAI](https://openai.com)
![Alt text](image.png)
```

<br><br><br><br><br>

## Inline Code and Code Blocks

Inline:

```
Use `git status` to check changes.
```

Block:

```
```bash
git status
git add .
```
```

<br><br><br><br><br>
