## What Is Obsidian?

Obsidian is a note app that stores everything as Markdown files.
That means your notes are:

- Plain text
- Portable
- Easy to version with Git

<br><br><br><br><br>

## The Obsidian Vault

A **vault** is just a folder of Markdown files.
Each note is a `.md` file.

You can open the same folder in any editor.
Obsidian does not lock you in.

<br><br><br><br><br>

## Linking Notes (The Best Feature)

You can link notes like this:

```
[[Git Basics]]
```

Obsidian turns that into a clickable link.
This makes your notes feel like a personal wiki.

<br><br><br><br><br>

## Tags and Search

Tags help you group ideas:

```
#git #notes #class
```

Obsidian also has powerful search, so you can find anything quickly.

<br><br><br><br><br>

## Summary

Markdown is not just for coding.
It is also perfect for personal notes and knowledge bases.
