## What is GitHub?

GitHub is a platform where developers store their code, collaborate with others, and contribute to open-source projects. It’s like Google Drive, but made for code.

## Git vs GitHub

- **Git** is the _tool_ for tracking changes.
- **GitHub** is a _place_ to store those changes online and collaborate.

Think of Git as Microsoft Word and GitHub as Google Docs.

## Why GitHub?

- Store your code online (cloud backup).
- Collaborate with others.
- Contribute to open-source.
- Share your work professionally (portfolio).

## GitHub for Students

GitHub is used by professionals all over the world. You’ll also get access to **GitHub Student Developer Pack** which gives you free tools for learning and building.
