## What is Clone?

Cloning a repo downloads it to your local machine.

```bash
git clone https://github.com/YOUR-USERNAME/REPO-NAME.git
cd REPO-NAME
```

## What is Pull?

Pulling means getting the latest changes from GitHub.

```bash
git pull origin main
```

## When to Use

- Clone when starting a new project.
- Pull when someone else has pushed changes.

⚠️ Note: Always pull before pushing to avoid conflicts.
