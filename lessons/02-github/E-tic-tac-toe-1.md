## Why This Project?

We will use this **Tic Tac Toe** game to demonstrate Git and GitHub in a real project.

## Setup

1. Create a directory:

```bash
mkdir tic-tac-toe
cd tic-tac-toe
```

2. Create `tic_tac_toe.py` file and paste the code from the lesson.

3. Initialize Git and commit:

```bash
git init
git add tic_tac_toe.py
git commit -m "Initial version of Tic Tac Toe"
```

4. Push to GitHub.

## Iterative Changes

We will make changes in small steps, like:

- Add comments

- Add winning message

- Refactor to functions

Each step will be a new commit. This is version control in action.
