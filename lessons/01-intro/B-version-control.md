## What is Version Control?

It is a system that records changes to files so you can recall specific versions later.

## Types of Version Control

- **Local VCS** – Track changes on your local machine.
- **Centralized VCS** – A central server (e.g., SVN).
- **Distributed VCS** – Everyone has a copy (e.g., Git).

## Why Version Control?

- Protect against data loss
- Collaborate safely
- Track history of changes

## Git is a Distributed VCS

Each user has the full history locally, making it faster and more reliable.
